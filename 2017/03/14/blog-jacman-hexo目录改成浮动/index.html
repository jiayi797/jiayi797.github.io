jacman/hexo目录改成浮动

发表于 2017-03-14 更新于 2018-12-17 分类于 瞎折腾

添加样式支持

为了不吧原先的像是文件搞得太乱，这里，添加子集的样式文件。 首先，在样式文件的source文件夹下找到css文件夹，打开style.styl文件，在最后添加：

@import "_my/mycss";

新建自定义样式

找到样式文件夹css 新建_my文件夹，在其中新建mycss.styl文件，之后就可以按照stylus的格式自定义样式了。

设置toc浮动

给mycss.sty添加：

#toc
 line-height 1.2em
 font-size 0.8em
 backgroud-color #fff
 float right
 position fixed
 right 30em
 top 20em

存在的问题

暂不支持自相应。
